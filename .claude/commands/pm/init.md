# /pm:init

Initialize CCPM (Claude Code Project Management) and configure GitHub integration.

## Usage
```
/pm:init
```

## Process

This command will:

1. **Verify Project Structure**
   - Check for `.claude/` directory
   - Create necessary subdirectories:
     - `.claude/prds/` - PRD storage
     - `.claude/epics/` - Epic storage
     - `.claude/issues/` - Issue tracking
   - Verify git repository

2. **Configure GitHub**
   - Check GitHub CLI installation
   - Verify GitHub authentication
   - Set up repository permissions
   - Configure API access

3. **Set Up Git Worktrees**
   - Create worktree structure for parallel work
   - Document worktree strategy
   - Set up branch naming conventions

4. **Create Initial Files**
   - Generate CLAUDE.md instructions
   - Create example PRD template
   - Document CCPM workflow
   - Set up progress tracking

## Output

- CCPM fully initialized
- GitHub integration ready
- All directories created
- Team can start immediate execution
